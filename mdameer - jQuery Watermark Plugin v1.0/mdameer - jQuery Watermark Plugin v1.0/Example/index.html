jQuery Watermark Plugin

Watermark is a beautiful hint within the input elements to explain the field data for users, and this jQuery plugin will do this for you easily


What this Can Do?
  • Each Watermark element have its own watermark text and CSS class
  • This plugin will not affect the form submission
  • Simple API, let you change value, watermark text and CSS class at runtime, and get jQuery watermark element, and the original element
  • Password field intelligent support
  • Very light weight — minified version ~ 2.3 KB

Help?

  • Initialization:
    • Constructor: $("#InputId").watermark({ watermarkText: "Watermark Text", watermarkClassName: "CSS Class Name" });
    • Parameters:
      • watermarkText: To set as input field text when it's empty
      • watermarkClassName: To set as input field CSS Class when it's empty
  • data:
    • watermarkElement: To Get the container for all methods to modify the watermark appearance (ex: $watermarkElement.data("watermarkElement")).
  • Methods:
    • val: To Set/Get the value, as the jQuery val() method (ex: $watermarkElement.data("watermarkElement").val("New Value")).
    • watermarkText: To Set/Get the watermarkText at runtime (ex: $watermarkElement.data("watermarkElement").watermarkText("New Watermark Text")).
    • watermarkClassName: To Set/Get the watermark CSS Class Name at runtime (ex: $watermarkElement.data("watermarkElement").watermarkClassName("New CSS Class Name")).
    • getWatermarkElement: To Get the watermark element as jQuery object (ex: $watermarkElement.data("watermarkElement").getWatermarkElement()).
    • getElement: To Get the original element as jQuery object (ex: $watermarkElement.data("watermarkElement").getElement()).
